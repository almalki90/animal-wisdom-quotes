# حكم حيوانية فلسفية 🌿✨

موقع يجمع أعمق الحكم المستلهمة من حياة الحيوانات وصراعها من أجل البقاء، يتم تحديثه يومياً بحكم جديدة تتسم بالبلاغة والعمق.

## رابط الموقع
[https://almalki90.github.io/animal-wisdom-quotes/](https://almalki90.github.io/animal-wisdom-quotes/)

## واجهة الـ API
يمكنك استخراج الحكم برمجياً عبر الرابط التالي (بصيغة JSON):
`https://almalki90.github.io/animal-wisdom-quotes/quotes.json`

### مثال على البيانات:
```json
{
  "id": 1,
  "quote": "القوة ليست في زئير يرجّ الأرض، بل في هيبة تجبر الغابة على الصمت قبل أن تطلّ.",
  "animal": "أسد",
  "source": "حكمة أسدية"
}
```

## التحديث اليومي
الموقع مجهز بسكربت يعمل عبر GitHub Actions ليضيف حكمة جديدة كل يوم بأسلوب فلسفي راقٍ.
